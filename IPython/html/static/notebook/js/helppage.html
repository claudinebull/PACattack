IPython Notebook Turtle

Description:

Ipython Notebook Turtle mimics the Python Turtle module.

Get Started

These lines will start up the module:

import pacturtle
T = pacturtle.pacturtle()

You may then add commands.

Example:

import pacturtle
T = pacturtle.pacturtle()
T.speed(7)
T.right(33)
T.pensize(4)
T.forward(70)
T.left(55)
T.pencolor("blue")
T.backward(100)



Figure 1: An example of the turtle commands being used in the IPython Notebook.

Press Play to start the turtle! Press P to pause/unpause.

Commands

T.speed(x)
Change the speed of the turtle to x. Currently there is no limit.
example: T.speed(10)

T.pensize(x)
Change the size of the pen to x. Currently there is no limit.
Example: T.pensize(10)

T.pencolor("x")
Change the color of the penstroke to x.
Example: T.pencolor("blue")

T.penup()
Lift the pen. It will not draw lines but the turtle will move.
Example: T.penup()

T.pendown()
Drop pen to canvas. It will draw the movement of the turtle.
Example: T.pendown()

T.isdown()
Check if the pen is down on canvas and drawing.
Example: T.isdown()

T.forward(x)
Moves the turtle forward by x pixels.
Example: T.forward(100)

T.backward(x)
Move the turtle backwards by x pixels.
Example: T.backward(100)

T.right(x)
Rotate the turtle clockwise by x degrees.
T.right(80)

T.left
Rotate the turtle counter-clockwise by x degrees.
T.left(90)

T.circle(x)
Make a circle with x radius. Currently not working as intended.
Example: T.circle(5)

T.goto(x, y)
Move turtle to coordinates (x,y)
Example: T.goto(70, 50)

T.home()
Move turtle back to start location
Example: T.home()
