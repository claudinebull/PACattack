Introduction

The IPython turtle project's goal was to modify Ipython so that a modified version of the turtle.py library could be ran. This was done by creating a python file that contained functions that returned text strings to the standard output. The client side then uses a mime type to identify the strings as turtle commands and uses a HTML canvas with paper.js to animate the turtle. The existing functionality would not show the movements of the turtle, only the end result.

Changes Made to the Code Base

mypaper.js is the javascript file that animates the turtle and draws on the canvas using paper.js (http://paperjs.org/)

outputarea.js also has moderate changes in it to facilitate the addition of the mime type and identifying turtle commands.

maintookbar.js and notebook.js were also edited to add buttons to the gui.

Anyone who wished to get involved can do so by contributing to the help and documentation of the project, as well as cleaning up and optimizing code.
